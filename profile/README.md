## MkDocs NG

[MkDocs](https://www.mkdocs.org/) is a great tool — but its upstream development has largely stalled.
Existing forks didn't quite hit the mark, so this organization exists to keep the project alive and moving forward.

This is a spare-time effort by someone who uses MkDocs and doesn't want to watch a good thing quietly fall apart.

### Projects

* [mkdocs](https://github.com/mkdocs-ng/mkdocs) — Active fork of MkDocs core

## Contributing 💪

If you rely on MkDocs and want to help keep it going, contributions of any size are very welcome —
bug reports, fixes, documentation, or just spreading the word.
